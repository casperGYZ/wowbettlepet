2:宠物组
  对战位置在霜火岭，主要目标是冰脊幼崽，后两只是飞行或者小动物所以需要一套配套阵容，很多人用的是灰猫+戴米多斯的仆从x2 但因为我只有一个戴米多斯外加上他身子板脆所以我放了一个吞噬者宝宝。
  三只都要是25级蓝色等级，如果灰猫能买到加速度的最好，他的技能先攻打两次。
  灰猫和戴米多斯可以拍卖行直接买，吞噬者宝宝是噬渊抓的，遍地是，取决于想不想折腾。
